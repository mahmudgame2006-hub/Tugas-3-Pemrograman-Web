Penjelasan Tugas 3 Pemrograman Web

Pada Tugas 3 Pemrograman Web, saya diminta untuk:

  • Membuat repository GitHub baru khusus untuk proyek website.
  • Mengupload file index.html, style.css, dan script.js.
  • Mengaktifkan fitur GitHub Pages pada branch main dan folder / (root).
  • Menguji website melalui URL yang dihasilkan oleh GitHub Pages.
